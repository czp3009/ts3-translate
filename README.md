# ts3-translate
TeamSpeak3 Google 机翻汉化(咕果造梗机)

输出的 `ts` 文件在项目的 `/google_translated_templates` 文件夹.

输出的 `qm` 文件在项目的 `/dist` 文件夹.

源文件来自 http://dl.4players.de/ts/client/translation/

如有雷同, 说明你也是机翻的.

# 如何使用
首先要有 [`TeamSpeak3`](https://www.teamspeak.com), 安装好之后, 打开软件根目录

将本仓库的 `/dist` 文件夹中的全部文件放入软件根目录的 `./translations` 文件夹.

重启软件后打开软件的设置界面 Tools->Options->Application->Language

调为中文后, 即可看到汉化效果.

救命(H)
